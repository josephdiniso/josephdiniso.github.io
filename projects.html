Anagrams Cheater

A Program Written to Give a Cheating Advantage in Anagrams


Using the Python DLIB library, I created a facial landmark detector using a pretrained model. The detector places six points around the eye in designated positions. Using these landmark points I then calculated the aspect ratio of the eye by taking the distance from the top of the eye to the bottom, as well as the distance from the left to the right. Then dividing the vertical distance by the horizontal I was able to get an aspect ratio. When the aspect ratio went below a certain threshold, for over five milliseconds, then a blink would be registered.

Before running the script, the user must input an argument with the letters given to them in an Anagrams game. The program then calculates the longest possible anagrams (with a six letter maximum) and displays them to the user with the longest (and most point lucrative) words appearing first. When the user blinks the program will then show the next word, allowing the user to input words as quickly as possible.
